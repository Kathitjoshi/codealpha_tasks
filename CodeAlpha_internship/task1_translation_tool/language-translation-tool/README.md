
# Language Translation Tool

A browser-based language translation application that translates text between 30 commonly used languages using the MyMemory Translation API. The project is implemented with plain HTML, CSS, and JavaScript and requires no build tools or package installation.

## Features

- Translation between 30 supported languages
- Character counter with a 1000-character input limit
- Language swap for both text and language selections
- Copy translated text to the clipboard
- Native browser text-to-speech for source and translated text
- Responsive layout for desktop and mobile devices
- No backend or local server required

- HTML5

- Web Speech API
- Font Awesome

## Execution

1. Extract the ZIP archive.
2. Open `index.html` in a modern web browser.
3. Ensure an active internet connection for translation requests.

## Limitations

- Translation quality depends on the public MyMemory API.
- The application requires internet access.
- Speech synthesis quality depends on the voices available in the user's browser and operating system.
- Public API rate limits may apply.
